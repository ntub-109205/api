#### `<GET> http://140.131.115.99/api/meme/show/{category_id}`

>* category_id值為1(梗圖)或2(長輩圖)

用途：顯示梗圖

* Parameters

|key |type             |
|----|-----------------|
|time|sometimes;boolean|

>* sometimes為選擇性加入參數

---

* Return Values

```yaml
{
    "meme": [
        {
            "meme_id": ${meme_id},
            "filelink": ${filelink},
            "author": ${author},
            "template_id": ${template_id},
            "count": ${梗圖的讚數量},
            "thumb": ${使用者是否按讚},
            "created_at": ${timestamp},
            "tags": (array) ${tags}
        },
    ]
}
```

---
