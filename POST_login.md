#### `<POST> http://140.131.115.99/api/login`

用途：登入

* Parameters

|key     |type           |
|--------|---------------|
|email   |required;email |
|password|required;string|

>* required為必要

---

* Return Values

```json
{
    "success": {
        "token": ${token}
    }
}
```

---

* Examples

``` html
http://140.131.115.99/api/login

email    = example@example.com
password = passwd
```
